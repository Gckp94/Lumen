# Font Files

This directory contains custom fonts for the Lumen application.

## Required Fonts

### Azeret Mono (for data display)
Download from: https://fonts.google.com/specimen/Azeret+Mono

Required files:
- `AzeretMono-Regular.ttf`
- `AzeretMono-Bold.ttf`
- `AzeretMono-Medium.ttf`

### Geist (for UI text)
Download from: https://vercel.com/font

Required files:
- `Geist-Regular.otf`
- `Geist-Bold.otf`
- `Geist-Medium.otf`
- `Geist-Light.otf`

## Installation

1. Download fonts from the links above
2. Extract the font files
3. Copy the required files to this directory

If fonts are not available, the application will fall back to system defaults.
